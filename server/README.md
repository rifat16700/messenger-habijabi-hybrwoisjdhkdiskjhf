# 🚀 Hybrid Engine — Signaling Server

Zero-cost P2P Chat & Calling App Signaling Server built with Node.js + Socket.io.

## Features
- WebRTC Signaling (1-on-1 & Group calls)
- Offline message buffering via file.io
- FCM push notifications
- Conference call with Tree/Sub-host routing
- Keep-alive ping support
